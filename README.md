# 项目说明

基于 NestJS 框架搭建的 OPENAI 代理接口

注：因为某些原因，需要搭建一个 OPENAI 的转发服务放在外网服务器上

```prot
ORIGIN=
OPENAI_ORGANIZATION=
OPENAI_API_KEY=
```
